# 🚀 **STEP 1: Install Required Tools**

## 1. Install Docker

```bash
sudo apt update
sudo apt install -y docker.io
sudo systemctl start docker
sudo systemctl enable docker
```

Fix permission:

```bash
sudo usermod -aG docker $USER
newgrp docker
```

Check:

```bash
docker --version
```

---

## 2. Install kubectl

```bash
sudo apt install -y kubectl
kubectl version --client
```

---

## 3. Install Minikube

```bash
curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
sudo install minikube-linux-amd64 /usr/local/bin/minikube
```

Check:

```bash
minikube version
```

---

## 4. Install Terraform

```bash
wget https://releases.hashicorp.com/terraform/1.6.6/terraform_1.6.6_linux_amd64.zip
unzip terraform_1.6.6_linux_amd64.zip
sudo mv terraform /usr/local/bin/
terraform -version
```

---

# 🚀 **STEP 2: Start Kubernetes Cluster**

```bash
minikube start --driver=docker
```

Verify:

```bash
kubectl get nodes
kubectl cluster-info
```

# 🚀 **STEP 3: Create Terraform Kubernetes Resource**

## 1. Create project

```bash
mkdir terraform-k8s
cd terraform-k8s
nano main.tf
```

---

## 2. Paste Terraform code

```hcl
provider "kubernetes" {
  config_path = "~/.kube/config"
}

resource "kubernetes_pod_v1" "nginx" {
  metadata {
    name = "nginx-pod"
    labels = {
      app = "nginx"
    }
  }

  spec {
    container {
      image = "nginx"
      name  = "nginx-container"
    }
  }
}
```

---

## 3. Run Terraform

```bash
terraform init
terraform apply -auto-approve
```

---

## 4. Verify

```bash
kubectl get pods
```

## 3. Commands inside MongoDB





# 🚀 **STEP 5: Create CI/CD Workflow**

```bash
mkdir -p .github/workflows
nano .github/workflows/terraform.yml
```

---

# 🚀 **STEP 6: Paste GitHub Actions YAML**

```yaml
name: Terraform CI/CD

on:
  push:
    branches:
      - main

jobs:
  terraform:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout Code
      uses: actions/checkout@v3

    - name: Setup Terraform
      uses: hashicorp/setup-terraform@v3

    - name: Terraform Init
      run: terraform init

    - name: Terraform Apply
      run: terraform plan
```
